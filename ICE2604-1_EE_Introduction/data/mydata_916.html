• In geometry, a simple path is a simple curve, namely, a continuous injective function from an interval in the set of real numbers R {\displaystyle \mathbb {R} } to R n {\displaystyle \mathbb {R} ^{n}} or more generally to a metric space or a topological space.
  • In graph theory a simple path is a path in a graph which does not have repeating vertices. See path (graph theory).
